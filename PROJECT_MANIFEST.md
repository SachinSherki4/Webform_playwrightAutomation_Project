## 🏗️ Architecture Overview

### Page Object Model (POM)
```
BasePage (Base functionality)
    ↓
FormPage (Form-specific interactions)
    ├── navigateToForm()
    ├── fillForm()
    ├── submitForm()
    ├── getSuccessMessage()
    ├── getErrorMessage()
    └── ... (15+ methods)
```

### Test Execution Flow
```
Test File
    ↓
Fixtures (validFormData, logger, etc.)
    ↓
FormPage Object (POM)
    ↓
Utils (validators, logger, data generator)
    ↓
Playwright Page
    ↓
Browser (Chromium, Firefox, WebKit)
    ↓
Web Application (localhost:3000)
```

### Reporting Pipeline
```
Test Results
    ↓
Execution Utils (collect test data)
    ↓
Reporting Utils (process results)
    ↓
Multiple Formats
    ├── HTML Report
    ├── JSON Report
    ├── JUnit XML
    ├── CSV Export
    └── Console Output
```

## 🚦 Quick Start (5 Minutes)

```bash
# 1. Install dependencies
npm install
npx playwright install

# 2. Configure environment
cp .env.example .env
# Update BASE_URL to http://localhost:3000/

# 3. Run tests
npm test

# 4. View report
npm run test:report
```
