## 🚀 Quick Start (5 Minutes)

### Step 1: Install
```bash
npm install
npx playwright install
```

### Step 2: Configure
```bash
cp .env.example .env
```

```bash
npm test
```

### Step 4: View Report
```bash
npm run test:report
```
